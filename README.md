# music_player 🎵

A lightweight torrent client designed for listening to your local music. 

## 📥 Download
You can download the latest executable version from the **Releases** section:
**[Click here to download music_player.exe](https://github.com/r1zon4ik/music_player/releases/latest)**

## ✨ Features
* Simple and clean interface.
* Support for local music libraries.

## 🚀 How to use
1. Download the `music_player.exe` file.
2. Run the application.
3. Enjoy your music!
